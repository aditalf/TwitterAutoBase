# Installation

Twitter Auto Base requires [PHP](https://www.php.net/) to run.
Edit file config.php before you run this script
```sh
$ git clone https://github.com/nthanfp/TwitterAutoBase
$ cd TwitterAutoBase
$ cd lib
$ composer install
$ cd ../
$ php AutoBase.php
```
